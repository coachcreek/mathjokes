📖 How the Worksheet Works

  1. Set up: Choose a math skill level and optionally a specific joke, then click Create Worksheet PDF.
  2. The code grid: The top of the worksheet shows a blank box per letter in the joke's answer, each with a code number below it.
  3. Solve the math: Each stacked equation in the worksheet has a letter label. Solve it to get the code number for that letter.
  4. Decode: Find the code number in the grid and write the letter in the box above it. Fill all the boxes to reveal the joke!
